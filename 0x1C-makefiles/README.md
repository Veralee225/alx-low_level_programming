# 0x1C-makefiles

## 0. make -f 0-Makefile

- Create your first Makefile.

### Requirements:

- Names of the executable: *school*
- rules: all

- The all rule build your executable
- variables: none

## 1.make -f 1-Makefile

- name of the executable: *school*
- rules: *all*
 - The *all* rule builds your executable
- variables: *CC*, *SRC*
    - CC: the compiler to be used
    SRC: the *.c* files
